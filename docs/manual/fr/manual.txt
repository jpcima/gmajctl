== Description

gmajctl est un contrôleur midi compatible avec le multieffet G-Major de TC Electronic.

Ce programme permet de rendre conviviale l'utilisation avancée du multieffet grâce à une interface graphique qui regroupe tous les paramètres des préréglages.

Il offre la possibilité de récuperer dans le G-Major des préréglages utilisateur, d'en créer de nouveaux, de modifier ceux qui existent déjà, de les importer, de les exporter...

Vous allez découvrir toutes les possibilités offertes par gmajctl en lisant ce manuel.

== Installation

Le projet met à disposition des téléchargements pour le système Windows. +
https://github.com/linuxmao-org/gmajctl/releases

À défaut de disposer d'une version téléchargeable pour son système d'exploitation, il faudra récupérer le code source et le construire soi-même.
Visitez alors la page d'accueil du projet, et suivez les étapes de construction indiquées dans le fichier Readme.md.

== Utilisation

À l'ouverture du contrôleur, la fenêtre affiche l'interface graphique où l'on retrouve tous les réglages qui sont physiquement disponibles sur le G-Major et les fonctions nécessaires au transfert des données vers le G-Major.
