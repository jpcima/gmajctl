== Description

gmajctl est un contrôleur MIDI compatible avec le multi-effets G-Major de TC Electronic.

Ce programme permet de rendre conviviale l'utilisation avancée du multi-effets grâce à une interface graphique qui regroupe tous les paramètres des préréglages.

Il offre la possibilité de récuperer dans le G-Major des préréglages utilisateur, d'en créer de nouveaux, de modifier ceux qui existent déjà, de les importer, de les exporter...

Vous allez découvrir toutes les possibilités offertes par gmajctl en lisant ce manuel.

=== Préréglages

- Sélectionnez un patch sur lequel vous voulez travailler dans la liste des patchs.
Si l’option Temps Réel est activée le G-Major chargera ce patch et changera de preset pour l’utiliser.
Change - Si l’option Temps Réel est désactivée, Ce bouton fait changer  G-Major de patch.
New – Crée un nouveau patch vide dans la position sélectionnée. Si un patch est présent sur cet emplacement il est détruit.
Copy to – Copie un patch existant sur la position sélectionnée. Si un patch est présent sur cet emplacement il est détruit.
Import – Importe un patch précédemment exporté ou téléchargé sur guitar patches
Export - Exporte les patchs en fichier .realpatch, qui peuvent etre uploadés sur guitar patches.

=== MIDI

- Le menu midi permet de sélectionner l’interface midi-out à utiliser.
La coche real time permet de basculer dans le mode temps réel. Ce mode permet d’appliquer les modification de manière instantanée.
Send – Ce bouton envoi le patch courant au G-Major quand le mode real time n’est pas sélectioné.

=== Paramètres préréglages

- Ici on règle le routing, les relais, le tap tempo, le volume du preset du patch. Pour change le nom du patch en cours, en mode temps réel, il suffit d’écrire le nouveau nom dans le champ patchname et appuyer sur entrée pour qu’il soit envoyé au G-Major.

== À propos

gmajctl est un logiciel développé par Jean Pierre Cimalando en 2018. +

Ce logiciel est réalisé avec un objectif&nbsp;: faciliter l'utilisation du multi-effets G-Major qui n'est pilotable que par les boutons présents en façade ou par injection d'un message Sysex.
